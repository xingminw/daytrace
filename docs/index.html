Your day, traced from the signals your AI tools already emit.

Codex, Claude Code, Cursor, DeepSeek — these are how your real work happens now. DayTrace is a local-first personal trace system that quietly collects the same signal your AI tools already emit, joins it to your local code and tasks, and turns it into a daily narrative you actually want to read.

Quick start

# Clone, install, point at your data sources.
git clone https://github.com/xingminw/daytrace
cd daytrace
make install

# Set up DeepSeek + Gmail credentials (optional but recommended).
mkdir -p ~/.daytrace
cat > ~/.daytrace/secrets.env <<'EOF'
DEEPSEEK_API_KEY=sk-...
DAYTRACE_GMAIL_USER=
DAYTRACE_GMAIL_APP_PASSWORD=
DAYTRACE_EMAIL_TO=
EOF
chmod 600 ~/.daytrace/secrets.env

# Run it.
make daily        # collect + summarize yesterday
make dashboard    # open http://127.0.0.1:8765/today
