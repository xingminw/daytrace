# DayTrace dashboard demo

This folder is a **static snapshot** of the DayTrace dashboard
rendered against synthetic data — no real activity is shown.

Regenerate with `python scripts/build_demo.py`.

Pages (English is the default; switch language via the 中/EN pills in the header):
- Weekly report (landing) — [English](index.html) · [中文](index.zh.html)
- Daily report — [English](daily.html) · [中文](daily.zh.html)
